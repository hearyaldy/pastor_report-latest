# Theme Customization - Quick Start Guide

## 🚀 Quick Access

### Enable Dark Mode (3 Steps)
1. Open **Profile** or **Settings**
2. Find **"Dark Mode"** toggle
3. Switch it **ON**

### Change App Color (3 Steps)
1. Go to **Profile** → **Appearance** → **Primary Color**
2. Pick a color from the palette
3. Done! Color applies instantly

## 💡 Pro Tips

1. **Battery Saving**: Use dark mode on OLED screens
2. **Readability**: Choose high-contrast colors for your environment
3. **Professional Look**: Stick with Navy Blue or Blue for formal settings
4. **Personal Touch**: Match your church/ministry brand colors
5. **Auto-Apply**: Changes save automatically - no manual save needed

## 🔧 Common Tasks

### Switch Between Themes Quickly
- Profile → Appearance → Dark Mode toggle
- One tap to switch, instant update

### Reset to Default
1. Go to Primary Color picker
2. Select Navy Blue (first option)
3. Theme returns to default

### Try Different Colors
- No commitment needed
- Switch as often as you like
- See changes immediately
- All screens update automatically
